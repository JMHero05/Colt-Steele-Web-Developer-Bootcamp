Conditional Exercises

  • If age is below 18
    • Display "Sorry, you are not old enough to enter the venue."

  • If age is between 18 & 21
    • Display "You can enter, but cannot drink."

  • If age is above 21
    • Display "Come on in. You can drink."

  • If age is negative
    • Display "You can't have a negative age. Get a better fake next time."

  • If age is 21
    • Display "Happy 21st birthday! LET'S RAGE!!!"

  • If age is odd
    • Display "Your age is odd...are you?"

  • If age is a perfect square
    • Display "Your age is a perfect square and I'm a nerd!"
